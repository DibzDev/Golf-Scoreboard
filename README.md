Golf Scoreboard

A futuristic, interactive golf scoreboard built with HTML, Tailwind CSS, and JavaScript.
Track scores for 18 holes, calculate results vs par, and export data in JSON format. Features light/dark mode, responsive layout, and modern UI design.

Features

Track 18 holes with par and strokes input.

Automatically calculate results: Hole-in-one, Eagle, Birdie, Par, Bogey, Double Bogey, Go Home!.

Summary cards: Total Score, Total Par, Score vs Par.

Light/Dark mode toggle with persistent preference in localStorage.

Responsive design for mobile, tablet, and desktop.

Export scoreboard data as JSON.

Futuristic UI with hover effects, smooth transitions, and neon-inspired highlights.
